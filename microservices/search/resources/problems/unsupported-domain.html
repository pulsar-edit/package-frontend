Type URI: https://search.pulsar-edit.dev/problems/unsupported-domain
Title: The Domain provided is not supported.
Recommended HTTP status Code: 400
Reference: RFC 9457
Details: The Pulsar search service only supports specific Pulsar subdomains, as defined in `https://github.com/pulsar-edit/package-frontend`/`microservices/search/index.js`. If the provided domain is not supported there it will not work in search until it is.
